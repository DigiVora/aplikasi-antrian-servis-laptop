# 💻 Antrian Servis Laptop (Queue - FIFO)

Aplikasi **Antrian Servis Laptop** merupakan program sederhana berbasis **C++** yang menerapkan konsep **Queue (First In First Out / FIFO)** sebagai implementasi dari materi **Praktikum Struktur Data**.

Program ini mensimulasikan sistem antrian pada sebuah tempat servis laptop, di mana pelanggan yang datang lebih dahulu akan dilayani lebih dahulu.

## 📚 Konsep Queue (FIFO)
Queue merupakan struktur data dengan prinsip **First In First Out (FIFO)**, yaitu data yang pertama masuk akan menjadi data pertama yang keluar.

Contoh:
```text
Masuk Antrian:
Yakin
Budi
Andi

Urutan Dilayani:
Yakin
Budi
Andi
```

---

## 🧩 Operasi Queue yang Digunakan
| Operasi | Fungsi |
|---------|--------|
| `push()` | Menambahkan pelanggan ke antrian |
| `pop()` | Menghapus pelanggan yang telah dilayani |
| `front()` | Menampilkan pelanggan paling depan |
| `empty()` | Mengecek apakah antrian kosong |
| `size()` | Menghitung jumlah pelanggan |
